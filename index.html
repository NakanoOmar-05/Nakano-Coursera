Bienvenido a las páginas de GitHub. Este generador automático de páginas es la forma más fácil de crear páginas hermosas para todos sus proyectos. Crea el contenido de tu página aquí usando GitHub Flavored Markdown, selecciona una plantilla creada por un diseñador y publica. Una vez generada su página, puede consultar la nueva rama 'gh-pages' localmente. Si usas GitHub Desktop, simplemente sincroniza tu repositorio y verás la nueva rama. ### Plantillas de diseñador Hemos creado algunas plantillas atractivas para que las uses. Continúe y haga clic en 'Continuar con los diseños' para navegar a través de ellos. Puede volver fácilmente a editar su página antes de publicarla. Después de publicar su página, puede volver a visitar el generador de páginas y cambiar a otro tema. El contenido de tu página se conservará. ### Creación manual de páginas Si prefiere no utilizar el generador automático, inserte una rama llamada 'gh-pages' en su repositorio para crear una página manualmente. Además de admitir contenido HTML regular, GitHub Pages admite Jekyll, un generador de sitios estáticos simple y consciente del blog. Jekyll facilita la creación de encabezados y pies de página en todo el sitio sin tener que copiarlos en todas las páginas. También ofrece soporte de blog inteligente y otras funciones avanzadas de plantillas. ### Autores y colaboradores Puedes @mention un nombre de usuario de GitHub para generar un enlace a su perfil. El elemento '' resultante se vinculará al perfil de GitHub del colaborador. Por ejemplo: En 2007, Chris Wanstrath (@defunkt), PJ Hyett (@pjhyett) y Tom Preston-Werner (@mojombo) fundaron GitHub. ### Soporte o Contacto ¿Tiene problemas con pages? Consulte nuestra [documentación] (https://docs.github.com/pages) o [póngase en contacto con el soporte] (https://support.github.com/contact) y le ayudaremos a resolverlo.
